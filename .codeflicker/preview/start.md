# my-codeflicker 启动指南

## 项目概述
基于 Vue 3 + Vite 构建的单页应用，使用 Pinia 状态管理，由单文件 HTML 迁移而来的桌面风格 UI 项目。

## 快速启动

```bash
npm run dev
```

**启动后访问**：http://localhost:5173

```yaml
subProjectPath: .
command: npm run dev
cwd: .
port: 5173
previewUrl: http://localhost:5173
description: Vue 3 + Vite 前端应用，myflicker 桌面 UI
```
