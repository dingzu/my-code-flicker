# 图片编辑功能说明

## 功能概述

在资产预览区域点击图片后，现在支持以下四种 AI 图片编辑功能：

### 1. 图生图
- **功能描述**：使用 AI 生成一张风格相似的新图片
- **参数配置**：
  - 相似度调节（0-100%）
  - 可选提示词输入

### 2. 局部重绘（布局重绘）
- **功能描述**：在图片上涂抹区域，并输入描述词优化该区域
- **参数配置**：
  - 画笔大小调节（5-100px）
  - 涂抹/擦除模式切换
  - 优化描述输入框
  - 清空蒙版功能

### 3. 图像扩展
- **功能描述**：修改图片尺寸，AI 自动扩展并填充新增区域
- **参数配置**：
  - 扩展方向选择（全部/上/下/左/右）
  - 扩展尺寸调节（50-500px）
  - 可选扩展提示词

### 4. 智能抠图
- **功能描述**：自动识别主体并移除背景
- **参数配置**：
  - 抠图模式（自动识别/手动标记）
  - 边缘精细度调节（0-100%）

## 使用流程

1. **打开图片预览**：在资产列表中点击图片类型的资产
2. **选择编辑工具**：在工具栏中点击对应的编辑按钮
3. **配置参数**：在右侧弹出的编辑面板中调整参数
4. **开始处理**：点击"生成图片"/"开始重绘"/"开始扩展"/"开始抠图"按钮
5. **等待结果**：AI 处理完成后会显示新生成的图片

## 技术实现

### 组件结构

```
AssetPreview.vue (主预览组件)
├── ImageEditToolbar.vue (图片编辑工具栏)
└── ImageEditPanel.vue (编辑参数面板)
```

### 文件类型检测

系统会自动检测资产文件扩展名，支持的图片格式包括：
- jpg/jpeg
- png
- gif
- webp
- svg

### API 集成点

在 `AssetPreview.vue` 的 `handleImageEdit` 方法中集成实际的 AI 图片编辑 API：

```javascript
function handleImageEdit(data) {
  console.log('Image edit:', data)
  // TODO: 调用 AI 图片编辑 API
  // data.tool: 工具类型
  // data.params: 编辑参数
}
```

## 样式特点

- 工具栏按钮使用浅色背景，激活状态为蓝色高亮
- 编辑面板采用毛玻璃效果，带有圆角和阴影
- 滑块和按钮交互有流畅的过渡动画
- 图片预览区域使用深色背景突出图片内容

## 后续扩展

可以在 `ImageEditPanel.vue` 中添加更多编辑功能：
- 图片滤镜
- 风格转换
- 图片修复
- 超分辨率
- 人像美化

## 注意事项

1. 目前图片预览使用的是 `asset.url`，需要确保资产对象包含有效的图片 URL
2. AI 图片编辑功能需要集成实际的后端 API 服务
3. 局部重绘功能的画布涂抹需要在 Canvas 上实现（当前为占位功能）
4. 建议添加加载状态和错误处理机制
