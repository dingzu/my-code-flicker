# My CodeFlicker Desktop

Myflicker 桌面端原型演示框架。

## 文件说明

- `myflicker-desktop.html` — 主界面原型，单文件 HTML，无需构建工具直接浏览器打开
- `public/1.jpg` — Myflicker 产品图标

## 核心功能演示

- macOS 窗口 Chrome（拖拽、缩放、交通灯按钮）
- Arc 风格侧边栏：项目（支持工作区绑定）+ 独立对话
- 对话框直接输入，无模式选择障碍
- Context pill 切换执行位置（本地工作区 / 云端 / 无项目）
- 新建项目 dialog（云端 or 本地工作区 + Git 检测）

## 使用

直接用浏览器打开 `myflicker-desktop.html` 即可预览。
